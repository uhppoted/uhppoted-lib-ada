function Activate_Keypads
  (U       : UHPPOTE;
   C       : Unsigned_32;
   Keypads : Uhppoted.Lib.Keypads;
   Timeout : Duration := 2.5) return Boolean

Activates/deactivates the keypads associated with a controller door card reader. Restricted to the local LAN.

Parameters
U

UHPPOTE configuration.

C

Controller serial number.

Keypads

List [1..4] of keypads to be activated/deactivated.

Timeout

Operation timeout (defaults to 2.5s).

Return Value

Raised exceptions
Timeout_Error

if the controller did not respond.
